# Нода песочницы

Нода Песочницы Mina позволит вам протестировать и ознакомиться с основными функциями протокола, а также построить инструментарий в стабильной среде - это приватная сеть с одной нодой, использующая ту же конфигурацию, что и реальная тестовая сеть. Песочница поддерживает несколько учетных записей, отправляя транзакции между ними, а также поддерживает выполнение SNARK-работы, делегирование и стейкинг. Более того, поскольку это сеть с одним узлом, вы получите все вознаграждения за блок!

<Alert>

Песочница **НЕ** подключает вас к действующей сети.

</Alert>

## Установка

[Докер](https://www.docker.com) это инструмент для портативных запущенных приложений. The Mina Sandbox Node is packaged with Docker. Установить его очень просто--мы предлагаем [Docker Desktop](https://www.docker.com/products/docker-desktop).

После установки Docker, запустите следующую команду, чтобы развернуть Песочницу Mina.

```
docker run \
  --publish 3085:3085 \
  -d \
  --name mina \
  -e RUN_DEMO=true \
  minaprotocol/mina-daemon:1.3.0-9b0369c-bullseye-devnet
```

Эта команда запустит daemon внутри контейнера докера и предоставит вашему компьютеру GraphQL-порт (3085), который используется для связи с клиентом. Daemon будет автоматически запущен в фоновом режиме с блок-продюсером и снарк-работником.

Вы можете просматривать логи, выполняя

```
docker logs --follow mina
```

И остановить Mina, запустив

```
docker stop mina
```

Вы можете использовать Mina CLI для взаимодействия с нодой песочницы. Следующая команда открывает оболочку внутри контейнера докера, откуда вы можете запустить любую из доступных <DocLink copy="команд Mina" url="/using-mina/cli-reference" />.

```
docker exec -it mina bash
```

## Как использовать Песочницу

Есть несколько вещей, которые вы теперь можете сделать с вашей песочницей, когда она работает:

- <DocLink copy="Установите Mina" url="/getting-started#installation" /> в обычном
  режиме и используйте многочисленные команды клиента. Так как daemon уже запущен
  в контейнере, вам не нужно запускать `mina daemon`!

- Установите приложение <DocLink copy="GUI Wallet" url="/gui-wallet" />, чтобы использовать графический интерфейс для вашей ноды. Введите '127.0.0.1' в качестве хоста вашей ноды во время установки.

- Зайдите на сайт http://localhost:3085/graphql, чтобы напрямую поиграть с GraphQL API.
