# Developers

Welcome to the Mina developer docs - please quickly familiarize yourself with the below links to get a lay of the land.

Explore the codebase on [Github](https://github.com/MinaProtocol/mina).

See the <DocLink copy="Contributing Guide" url="/developers/contributing" /> to start contributing code to Mina. The [protocol](https://github.com/MinaProtocol/mina/tree/master/src) and [CLI](https://github.com/MinaProtocol/mina/tree/master/src/app/cli) are written in OCaml. Any level of experience in any or all of these tools is welcome.
